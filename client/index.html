PresidentAPI - Overview

President API gives an opportunity to gain information about all presidents in the World by typing his or her full name

  • President's full information
  • High quality images
  • Interaction with Google Maps Api

We provide the following types of API request

  • President Request is GET HTTP request returns the chosen president's information
  • Country Request is GET HTTP request return the chosen country's information
  • Countries Request is GET HTTP request return all countries' information

There is not Terms of Service, these are things you should know

  • All API requests are free, no limit for you
  • No CORS policy - all IPs are welcome
  • Only GET HTTP request is available - do not change our DB

If you have any questions, you can check the FAQ, choose in on the left panel

PresidentAPI - API Request Types

President Request

https://president-api.appspot.com/api/countries/[country]/[president]

President Request returns the full data about the chosen president. You will see object with properties, which contains the specific data

Example Request
GET https://president-api.appspot.com/api/countries/us/Donald-Trump

Request Parameters

Required parameters slot directly into the request URL

  • country required
    The abbreviation of country to move inside specific country.
    This is a list of available countries for right now:
    • us stays for USA
    • ca stays for Canada
    • cn stays for China
    • ru stays for Russia
    • br stays for Brazil
  • president required
    The full name of the chosen president. You need to type it in a specific format to get a right response

    The string format: Firstname-Lastname as Vladimir-Putin

Country Request

https://president-api.appspot.com/api/countries/[country]

Country Request returns the full data about the chosen country. You will see the object with brief description of the country add array of presidents

Example Request
GET https://president-api.appspot.com/api/countries/us

{
    "name": "United States of America",
    "flag": "google.cloud/country1/flag/flag1.jpeg",
    "foundationDate": "July 4, 1776",
    "abbreviation": "us",
    "presidents": [ ... ] // 5 items
}

Required parameters slot directly into the request

  • country required
    The abbreviation of country to move inside specific country.
    This is a list of available countries for right now:
    • us stays for USA
    • ca stays for Canada
    • cn stays for China
    • ru stays for Russia
    • br stays for Brazil

Countries Request

https://president-api.appspot.com/api/countries

Countries Request returns the full data of this API. It consists of one array of 5 countries with array of presidents inside each country.

There are no required parameters for this type of request

PresidentAPI - API Response Format

API responses consist of a UTF-8-encoded, JSON-formatted object within the following properties

Country Object properties

  • name required
    The name of the country
  • flag required
    The image of country's flag
  • countryCoordinates deprecated
    The coordinates of the chosen country. Was deprecated because of the pointless
  • foundationDate required
    The date, when country was founded
  • demographics in development
    The demographics of the chosen country
  • abbreviation required
    The abbreviation of the country's name. Important property. The information is obtained, using this property in GET HTTP request
  • presidents required
    The array of presidents of chosen country

President Object properties

  • id required
    The Identification Number of a president (for dev purposes)
  • firstName required
    The first name of the president
  • lastName required
    The last name of the country
  • dOB required
    The date of birthday of the president
  • prose required
    The brief information of the president
  • timeOnPosition in development
    The time, which the president obtained the position
  • precededBy required
    The full name of the president, by who the chosen president was preceded
  • succeededBy required
    The full name of the president, by who the chose presided was succeeded
  • website required
    The website of the chosen president
  • wikipediaArticle required
    The Wikipedia Article about the chosen president
  • photos required
    The photos of the chosen president. It is array of 3 objects with photo's URLs inside of each object
  • whiteHouseMapCoordinates required
    The coordinates of the White House, where the president was located. It is an object with two properties: X and Y. This property can be used to interact with Google Maps API and other Maps libraries.

PresidentAPI - FAQ

General

What technologies did you use?

Node.js (Express), Mongoose (v.4.10.8), MongoDB, Babel, Tunnel-ssh (deprecated) PuTTY for tunneling (redirect SSH to localhost)

What services of GCP did you use?
  • App Engine to host the Web API
  • Compute Engine to run VM with Mongo Database
  • Google Cloud Storage to keep all photos
How interact with Google Maps API?

You just need to create your own project on Google Developers, then enable Google Maps JavaScript API. Then, you send GET HTTP request to the chosen president, and retrieve whiteHouseCoordinates from the JSON objects. After this, you put it inside the initMap() function
